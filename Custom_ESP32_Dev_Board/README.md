# Custom ESP32 Module

## Introduction
As a personal project I designed a custom ESP32 Module as a way to challenge to my PCB design skills.

## Design Highlights:

- ESP32 S3 chip
- 5V-30V in via pins
- 4 builtin leds (3 programable)
- USBC for power and/or programming
- A cool cat silkscreen
